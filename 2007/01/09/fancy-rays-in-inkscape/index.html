Fancy Rays in inkscape

A really nice thing to do with inkscape is to create rays. Rays are hot, maybe not as hot as swirly curls, but certainly worth your time.

In this tutorial I introduce the concept of gradients, and how to use duplicate tools to create repeating patterns. Inkscape is one of the best tools when you want to work with gradients. I think not even Adobe Illustrator can beat the gradient tools of Inkscape.

For rays, or any more digital drawing, we can best use inkscape in pixel-mode. Simplest to get there, is to use a template that is meant for the screen.

  1. create new: 1042 x 800

  2. Because we need to be precise, we add some guidelines. Click on one of the the rulers and drag a new guide onto the canvas. Doubleclick on a ruler to set its position in detail.

  3. Set File » Document properties, Snap to guides, so that we can draw precicely on the guides.

  4. Choose Bezier curves (Shift + F6) draw a triangle (use guides to make it exact)

  5. Fill it with a solid color, set 'stroke paint' to none.

  6. Toggle the selection mode to rotate mode, then drag centre crosshair to bottom corner.

  7. Now we duplicate this. A linked duplicate means that if you change the parent, all its linked 'children' follow that. You change the color of the parent: all children will change the color too. Linked duplicate of top-triangle (Alt + D), rotate 90° CCW Linked duplicate of top-triangle (Alt + D), rotate 90° CW Linked duplicate of top-triangle (Alt + D), rotate 90° CW twice

  8. Now that we have a nice cross, select all four triangles and group them.

  9. We now create a linked duplicate of the entire cross (Alt + D)

  10. Click the cross again (toggle into rotate mode). Hold CTRL for controlled rotation.

  11. Repeat the duplicating and rotating until you have the rays as dense as wished.

  12. Now select all the crosses and group them.

  13. Optionally add a coloured square behind the rays, you may want to paint them white, so then they won't be visible against the white canvas.

  14. Choose Stroke and paint (Ctrl + Shift + F) tool, select radial gradient. Click the 'Edit' button. Set both gradient ends to white (or whatever color you want the rays to be). But one end fully transparent.

  15. Now we make the gradient a bit more ray-ish. Add a new stop. Drag offset to +/-0.70 Drag alpha channel to +/-80

  16. With a nice image (download example here) in front, the rays come out really nice.

  17. For that, use the import tool (File » Import), and put it on top of the image.

Et voilá, a fancy schmancy sunset!
